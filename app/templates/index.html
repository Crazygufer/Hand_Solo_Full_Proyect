Handi Project

Descripción del Proyecto

Este proyecto involucra la creación de un brazo robótico de 5 grados de libertad controlado por comandos de voz. Utiliza tecnología avanzada como Whisper para la transcripción de voz y OpenCV para la visión artificial.
